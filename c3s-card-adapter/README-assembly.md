# OreSat C3 Surrogate to Card Adapter Board

# Description

The OreSat C3 Surrogate to Card Adapter Board is a 
19.4 x 34.6 x 1.6 mm four layer PCB with SMT components on
the top side.

This board adapts the C3S card to any other card, allowing you
to power and interface with that card. It's a "micro flatsat".

# Printed Circuit Board Assembly (PCBA) Information

## Assembly info

- Assemble to IPC Class 2 or better
- Bake components that are not moisture sealed to appropriate levels as required.
- Solder paste can be ROHS or leaded.
- No clean, or Aqueous flux and wash, can be used.
- No conformal coating.

# OreSat C3 Card Debug Board v3

# Description

The OreSat Card Debug Board v3 C3 is a 54.0 x 64.0 x 1.6 mm
 four layer PCB with SMT components on the top side.

This board allows users to power, debug, and communicate over serial with
OreSat cards. It requires a cable assembly to connect to a card.

# Printed Circuit Board Assembly (PCBA) Information

## Assembly info

- There are 3 fiducials on the top layer.
- Assemble to IPC Class 2 or better
- Bake components that are not moisture sealed to appropriate levels as required.
- Solder paste can be ROHS or leaded.
- No clean, or Aqueous flux and wash, can be used.
- No conformal coating.
